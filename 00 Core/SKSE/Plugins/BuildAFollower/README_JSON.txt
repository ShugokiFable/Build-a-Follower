Build a Follower — JSON save folders
====================================

Config:  Data\SKSE\Plugins\BuildAFollower\BuildAFollowerConfig.json  (created by MCM Save)
NPCs:    Data\SKSE\Plugins\BuildAFollower\BuildAFollowerNPCs\<Name>.json

RaceMenu presets are NOT stored here — they live under:
  Data\SKSE\Plugins\CharGen\Exported\*.jslot
  Data\SKSE\Plugins\CharGen\Presets\*.jslot
External head meshes (optional): same name as preset, .nif extension, same folders.

Upgrading from Stamp Followers? You can copy your old
  Data\SKSE\Plugins\StampFollowers\StampFollowersNPCs\*.json
into BuildAFollowerNPCs\ — preset, name, race, spells and gear carry over.
Voice and combat style referenced the old plugin and must be re-picked once.
